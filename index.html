Using Color in Web Development

Color Wheels


Every web developer need to become fluent in color theory and the concept of a color wheel. The first color wheel was developed by mathematician and physicist, Isaac Newton in 1665. During that year, a wave of bubonic plague swept through England, and as a result, Cambridge University temporarily closed. Newton returned home to Woolsthorpe, England and over the next months he changed the world with his theories on calculus, gravitation, and optics. In his famous prism experiment, he discovered that white light is composed of a spectrum of seven visible colors, which are red, orange, yellow, green, blue, indigo, and violet.

Subtractive Color Mixing


Newton organized the visible spectrum of colors into a color wheel with red, yellow, and blue as the primary colors which can be mixed to form secondary colors of orange, green, and purple. These secondary colors can be mixed to form tertiary colors of yellow-green (chartreuse), green-blue (teal),and blue-purple (violet). This type of color mixing, called subtractive mixing, only works when the source of light is being reflected, as it is in paint. Newton's color wheel can be seen in the image below:

Additive Color Mixing


When light is being emitted, as it is in a computer monitor, a different type of color mixing is required which is called additive mixing. The color wheel for additive color mixing is based on three primary colors of red, green, and blue, which can be mixed to form secondary colors of yellow, magenta, and cyan. These secondary colors can be mixed to form tertiary colors of red-yellow (orange), yellow-green (chartreuse green), green-cyan (spring green), cyan-blue (azure), blue-magenta (violet), and magenta-red (rose). Color Codes


Computer monitors are made up of an array of pixels, and each pixel is made up of three subpixels that emit red, green, or blue light. By adjusting the intensity of light emitted by each subpixel with an 8-bit code ranging from 0 - 255, over 16 million (2563) different additive color combinations are possible. The three hex codes can be specified with RGB, Hex, or HSL values.

RGB


RGB color codes are written as rgb(red, green, blue) where each parameter (red, green, and blue) defines the intensity of color with a value between 0 and 255. For example, rgb(255, 0, 0) displays as red, because the red subpixel is set to its highest value (255), while the other two subpixels, green and blue, are set to their lowest value (0):


rgb(255, 0, 0)

Hex


Hex color codes are written as hexidecimal or hex values #RRGGBB where RR (red), GG (green) and BB (blue) are hexadecimal integers between 00 and ff. For example, #00ff00 displays as green, because the green subpixel is set to its highest value (ff), while the other two subpixels, red and blue, are set to 00:


#00ff00

HSL


HSL Color codes are written as hsl(hue, saturation, lightness) where hue is expressed as degrees on the RGB color wheel from 0 to 360 (0 is red, 120 is green, and 240 is blue), saturation is expressed as a percentage from 0 to 100 (0% is gray, 50% is half gray / half color, and 100% is full color), and lightness is expressed as a percentage from 0 to 100 (0% is black, 50% is neither light nor dark, and 100% is white.) For example, hsl(240, 100%, 50%) displays as blue, because the hue for blue is 240 degrees, the saturation for full color is 100%, and the lightness for neither light nor dark is 50%.


hsl(240, 100%, 50%)

Opacity


All three of these color codes can be extended with an alpha parameter that specifies the opacity of the color. The color codes are then computed with the following formula:

adjusted color = α × foregroundColor + (1 - α) × backgroundColor



RGB color codes can be extended as rgba(red, green, blue, alpha) and HSL color codes can be extended as hsla(hue, saturation %, lightness %, alpha) where alpha is a decimal value between 0.0 (fully transparent) and 1.0 (fully opaque.) Hex color codes can be extended as #RRGGBBAA where AA is a hex value between 00 (fully transparent) and ff (fully opaque.)

Contrast Ratio


When using color, it's important to make sure the text color and the background color have sufficient contrast for readability. A contrast of at least 4.5:1 is recommended for regular size text and 3:1 for large text (18px or more). However, it's also important to avoid too much contrast. For example, the colors black and white have a contrast ratio of 21:1, which can also make reading more difficult. A good color combination usually has a contrast ratio around 10:1.
